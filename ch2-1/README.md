## C++ 챕터 2-1 실습과제 1
1. 전처리 또는 선행처리 단계를 설명하라
    - 전처리(Preprocessing)란 원시 데이터를 모델이 처리하기 적합한 형태로 변환하는 과정인데, 데이터 품질 개선과 모델 성능 향상과 처리속도 개선에 목적이 있다.(머신러닝, 딥러닝, 이미지 처리, 자연어 처리 등)

2. 객체 cin, cout은 어디에 선언되어 있는가? 선언된 파일을 찾아서 의미를 설명하고 캡쳐하여 첨부하라 -> 교재 75페이지 2.5절 내용을 참고할 것
    - 표준 입출력 객체로, iostream 헤더에 선언되어 있다.
    - 마이크로소프트 비주얼 C++(MSVC) : C:\Program Files (x86)\Microsoft Visual Studio\<버전>\VC\Tools\MSVC\<버전>\include\iostream


3. 프로그램을 링크할때 이름 충돌이 발생하는 경우를 설명해보라
    - 전역변수가 충돌할때 : 두개의 전역변수가 동일한 이름으로 정의되어있을때 링커가 어느 변수를 사용해야할지 몰라서 오류를 일으킴 
    - 전역 함수가 충돌할때 : 전역 함수 이름이 같을때 링커가 충돌을 보고 오류 발생
    - (네임스페이스를 사용하면 같은 이름이라도 다른 네임스페이스 안에 있기때문에 충돌하지 않음)

4. 스트림이란 무엇인가?
    - 데이터가 흐르는 통로를 추상화한 개념(데이터를 읽거나 쓰기 위해 연속적으로 흘려보내는 흐름을 의미)

5. 표준 출력 스트림을 정의할 때 사용하는 C언어 자료형은?
    - C++의 스트림 클래스는 C 언어에는 직접 대응되는 자료형이 없지만, 내부적으로 C라이브러리의 파일 구조체를 기반으로 구현된다.

6. 표준 출력 스트림을 정의할 때 사용하는 C++언어 자료형은?
    - cout : C++에서 표준 출력 스트림이다
    - 선언 위치는 iostream 헤더파일 안에서
    namespace std{
        extern ostream cout;
    }
    - 여기서 자료형은 std::ostream
