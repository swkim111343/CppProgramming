1. 메모리 누수를 설명하라.
    - 프로그램이 동적으로 할당한 메모리를더 이상 사용하지 않는데도 해제하지 않는 현상
    - 사용할 수 없는 메모리가 계속 쌓이는 상태

2. 메모리 누수가 왜 심각한 문제를 발생시키는지 설명하라.
    - 메모리 누수는 단순한 실수가 아니라 시스템 전체에 영향을 줌
    - 메모리 고갈, 성능 저하, 시스템 불안정, 디버깅 어려움

3. 메모리 누수를 방지하는 방법을 설명하라.
    - 할당과 해제를 습관화 한다.
    - 도구 사용(비주얼스튜디오 메모리검사, valgrind(리눅스))

4. C++ 스마트 포인터 클래스에 대해 조사하라
    - unique_ptr : 단 하나의 소유자만 가능, 복사 X 이동만 가능
    - shared_ptr : 여러 개가 같은 객체 공유
    - weak_ptr : shared_ptr의 순환 참조 문제 해결용

## 실습과제 3 
- 포인터 증가로 원래 주소를 잃어버림
<pre>Circle* pArray = new Circle[3];
...
pArray++;</pre>
- pArray는 배열의 시작 주소를 가지고 있어야 하는데 pArray++ 하면 주소가 계속 이동함
->pArray는 절대 변경하지 않고 포인터를 따로 사용한다
<pre>Circle* pArray = new Circle[3];
Circle* temp = pArray;

for (int i = 0; i < 3; i++) {
    cout << temp->getArea() << '\n';
    temp++;
}

delete[] pArray;</pre>
