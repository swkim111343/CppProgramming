## C++ 챕터 4-1 실습과제 1
1. 다음 빈칸을 채우시오. 정적(static)변수는 인터넷을 검색하여 조사해볼 것
    - 정적변수(static) : 위치는 전역변수랑 같음 (데이터 영역), 하지만 범위(scope)는 제한됨 (함수 내부 또는 파일 내부), 값이 계속 유지됨 (초기화 1번만)

| 변수의 종류        | 메모리 영역       | 메모리 할당 시점       | 메모리 해제 시점        |
| ------------- | ------------ | --------------- | ---------------- |
| 지역변수          | 스택(Stack)    | 함수 호출 시         | 함수 종료 시          |
| 전역변수          | 데이터 영역(Data) | 프로그램 시작 시       | 프로그램 종료 시        |
| 정적변수 (static) | 데이터 영역(Data) | 프로그램 시작 시       | 프로그램 종료 시        |
| 동적메모리         | 힙(Heap)      | malloc/new 호출 시 | free/delete 호출 시 |

## 실습과제 2 
1. 매크로상수 NULL이 무엇인지 조사하라, C/C++언어에서 정의의 차이, 정의된 헤더파일, 사용용도 등
    - 매크로 상수 : 포인터가 아무 것도 가리키지 않음을 나타내는 값
    - C / C++에서 정의 차이 : C는 void*를 허용하지만, C++은 타입이 엄격해서 단순히 0으로 정의됨
    - 정의된 헤더파일 : C: <stdio.h>, <stdlib.h>, <stddef.h>, C++: <cstddef>
    - 포인터 초기화 : int *p = NULL;
    - 함수 실패 표현 : if (p == NULL)

2. 널문자와 NULL의 차이를 설명하시오, 정의, 자료형, 메모리크기, 실제값, 사용용도 등

| 구분     | 널문자 (`'\0'`)    | NULL                    |
| ------ | --------------- | ----------------------- |
| 정의     | 문자열의 끝을 나타내는 문자 | 아무것도 가리키지 않는 포인터        |
| 자료형    | `char`          | 포인터 (`int*`, `char*` 등) |
| 메모리 크기 | 1바이트            | 4바이트 또는 8바이트 (시스템 의존)   |
| 실제 값   | 0               | 0 (주소값 0)               |
| 사용 용도  | 문자열 종료 표시       | 포인터 초기화 및 검사            |

3. 요즘은 NULL 대신에 nullptr을 사용하는데 nullptr은 무엇인지 조사하시오
    - C++에서 도입된 “진짜 포인터 전용 null 값”
    - 타입 : std::nullptr_t
