## C++ 챕터 3-2 실습과제 1
1. C++에서 생성자를 통하여 멤버변수의 초기화를 하는 이유를 설명하라.
    - 객체가 생성될 때 올바른 상태를 보장하기 위해서이다.
    - 객체의 일관성과 불변성 유지

2. 생성자 작성시 생성자 몸체에서 멤버를 초기화하는 것 보다 멤버 초기화 리스트에서 초기화하는 이유를 설명하라.
    - 몸체에서 멤버를 초기화하는 방식은 x생성, x에 초기화하는 방식 vs 초기화 리스트에서 멤버를 초기화하는 방식은 x를 10으로 바로 생성하기 때문에 효율적이다.
    - 즉 불필요한 과정을 제거하고 성능을 높히기 위해서이다.

3. 위임, 타겟 생성자로 나누어 작성하는 이유를 설명하라
    - 코드 중복을 제거하기 위해서이다.
    - 유지보수가 쉬움

4. 함수 몸체(body)에서 생성자를 직접 호출하면 어떻게 되는지 조사해보라
    - 예를들어 아래의 코드를 실행했을때 결과는 생성자 함수 처럼호출이 불가능하고 컴파일 에러가 발생한다.
