 ## C++ 챕터 5-2
1. 참조에 의한 호출과 주소에 의한 호출의 차이점을 설명하라.
-참조에 의한 호출
 변수의 별명(참조자) 을 함수에 전달하는 방식
 함수 안에서 원본 변수를 직접 사용함
 C++에서는 & 사용
<pre>
void swap(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}
</pre>

-주소에 의한 호출 
  변수의 메모리 주소(pointer) 를 전달하는 방식
  함수에서 포인터를 통해 원본에 접근함
  *, & 연산자를 사용
    
<pre>
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}
</pre>

2. 참조에 의한 호출의 장점은 무엇인가?
    - 원본 데이터 수정 가능
    - 복사 비용 감소
    - 코드가 간단하고 가독성이 좋음
