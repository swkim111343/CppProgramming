## C++ 챕터 3-3 실습과제 1
1. 객체의 생성과 소멸의 의미를 설명하라
    - 객체 생성 : 메모리에 객체를 위한 공간이 할당됨, 생성자(Constructor)가 호출되어 객체 초기화 수행
    - 객체 소멸 : 객체가 사용 끝나면 메모리에서 제거됨, 소멸자(Destructor)가 호출되어 정리 작업 수행

2. 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점, 소멸시점, 초기화 여부 등의 관점에서 비교하라

| **구분** | **지역객체(Local)** | **전역객체(Global)** |
|---|---|---|
| **선언위치** | 함수 내부 | 함수 외부(파일 상단) |
| **접근범위** | 선언된 블록 내부 | 전체 파일 |
| **생성시점** | 함수 호출 시 | 프로그램 시작 시 |
| **소멸시점** | 함수 종료 시 | 프로그램 종료 시 |
| **초기화** | 초기화 안하면 쓰레기 값 | 자동으로 0으로 초기화 |

3. 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라
    - 프로그램 실행 → main() 전에 전역 변수 초기화, 정적(static) 변수 초기화, 전역 객체 생성, 런타임 환경 설정이 수행됨

4. main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라
    - main() 종료 → 프로그램 종료 전 지역 객체 소멸, 전역 객체 소멸, 정적 객체 소멸이 수행됨
