## C++ 챕터 2-2 실습과제 1
1. 라이브러리는 헤더파일과 목적파일로 구성되어 있다. 헤더파일과 목적파일에 들어 있는 내용이 무엇인지 각각 설명하시오.
    - 헤더파일(.h) : 라이브러리에서 인터페이스를 정의하는파일(외부에서 라이브러리를 사용할때 어떤 함수, 클래스 등을 사용할 수 있는지 알려줌)
    - 목적파일(.obj) : 라이브러리의 실제 구현 코드를 담고 있는파일(기계어 코드, 링커가 결합할때 사용)

2. 함수의 선언, 정의, 호출은 각각 어디에 존재하는가?
    - 함수의 선언 : 헤더파일, 소스 파일 상단
    - 함수의 정의 : 소스파일, 라이브러리의 목적파일에 포함
    - 함수의 호출 : 메인함수 안

3. <iostream> 헤더파일에 cin, cout은 extern 변수로 선언되어 있다. extern 선언과 변수선언의 차이를 설명하라. 그리고, 그럼 cin, cout의 변수선언은 어디에 있는가?(교재 2.5절 참고)
    - 먼저 변수선언은 메모리를 할당하고 변수를 생성하는 것을 말한다(ex_int a=10;)(변수의 선언, 정의, 메모리 할당)
    - extern 선언은 외부에 이미 존재하는 변수를 사용하는것, 메모리를 할당하지 않음(이름과 타입만)(ex_extern int a;)

4. 라이브러리 함수나 객체를 사용하기 전에 헤더파일을 include하는 이유를 설명하라.
    - 함수와 객체를 제공하기 위해서이다
    - 예를들어 <iostream>에는 cout cin이 extern으로 선언되어있음, 헤더를 포함하면 컴파일러는 cout과 cin이 ostream/istream타입 객체임을 알 수 있다.
