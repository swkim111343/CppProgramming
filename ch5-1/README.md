## C++ 챕터 4-4 
1. 인자와 매개변수를 구분하여 설명하시오.
    - 인자 (argument) → 함수를 호출할 때 전달하는 값 → 예: func(10); 에서 10
    - 매개변수 (parameter) → 함수 정의에서 값을 받는 변수 → 예: void func(int x) 에서 x

2. 함수 호출시 자동으로 처리되는 2가지는 무엇인가?
    - 1.인자의 값이 매개변수로 복사됨 (값 전달)
    - 2.호출한 위치의 주소가 저장됨 (복귀 주소 저장)

3. 값에 의한 호출로 다른 함수에서 선언된 변수의 값을 변경할 수 없다. 이유를 설명하라.
    - 함수 호출 시 값이 복사되어 전달됨
    - 즉, 원본 변수와 완전히 다른 메모리 공간 사용
    - 함수 내부에서 값 변경 → 복사본만 변경됨

4. 주소에 의한 호출을 사용해야하는 경우를 2가지 이상 설명하라.
    - 1.함수에서 원본 값을 변경해야 할 때 → 예: swap 함수
    - 큰 데이터(배열, 구조체 등)를 효율적으로 전달할 때 → 복사 비용 줄이기

5. 인자가 객체인 경우는 어떤 호출방식이 좋은가? 이유도 설명하라
    - 주소에 의한 호출 (참조 or 포인터)
    - 객체는 크기가 크기 때문에 → 값 복사 시 비효율적 (속도/메모리 낭비)

6. 함수 반환과정에서 자동으로 처리되는 2가지는 무엇인가?
    - 반환값이 호출한 곳으로 전달됨
    - 저장된 복귀 주소로 돌아감

7. 함수 반환과정에서 임시변수가 필요한 이유를 설명하라
    - 함수 내부 변수는 함수 종료 시 소멸됨
    - 반환값을 바로 넘기면 사라질 위험 있음
    - 따라서 임시 공간(레지스터 또는 임시 변수)에 저장 후 전달
